# azure_ai_search

Para informações mais detalhadas acesse [Explore an Azure AI Search index (UI)](https://microsoftlearning.github.io/mslearn-ai-fundamentals/Instructions/Labs/11-ai-search.html)

## Criar Recursos no Azure
- Acesse o portal do Azure.
- Crie um recurso de Azure AI Search com configurações específicas
- Crie um recurso de Azure AI services alinhado com o recurso AI Search.
- Crie uma conta de armazenamento com contêineres de blobs para armazenar documentos brutos.

## Extrair e Carregar Dados:

- Extraia dados das avaliações do café e salve-os em uma pasta.
- Faça o upload dos dados extraídos para a conta de Armazenamento do Azure.

## Indexar os Documentos:
- Use o Azure AI Search para criar um índice e um indexador.
- Configure habilidades cognitivas (como OCR, análise de sentimento, extração de palavras-chave) para enriquecer os dados.

## Consultar o Índice:
- Utilize a ferramenta Search Explorer para escrever e testar consultas.
- Revisar o Conhecimento Armazenado:
- Explore o conhecimento armazenado para ver os dados enriquecidos.

Esse processo permite um estabelecimento analisar as avaliações dos clientes podendo fornecer insights valiosos sobre a experiência do cliente, incluindo análise de sentimento, temas principais e reclamações ou elogios comuns, identificando assim problemas recorrentes nos feedbacks dos clientes e isso ajuda o estabelecimento a fazer melhorias operacionais, refinar ofertas de produtos ou aprimorar o atendimento ao cliente.
